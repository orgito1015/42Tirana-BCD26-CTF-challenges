### Challenge 1 — 42: Find the Location (10 pts)

**Question:** Find where 42 is in Australia.

**Flag format:** `flag{city}`

#### Technical Analysis

The 42 network is a global group of tuition-free programming schools. Use the official 42 website or campus directory to locate the Australian campus.

#### Solving Approach

1. Navigate to [https://www.42network.org/campuses/](https://www.42network.org/campuses/) or [https://42.fr](https://42.fr)
2. Filter by country: Australia
3. Note the city where the campus is located

#### Expected Answer

`flag{<city_name>}` (the city name in lowercase)

### Challenge 2 — 42 Tirana: IT Administrator (10 pts)

This challenge asks for the username or alias used by 42 Tirana's IT Administrator in an online context. Possible sources include LinkedIn, GitHub, or the 42 intranet profile system.

#### Solving Approach

1. Search LinkedIn for the IT Administrator at 42 Tirana
2. Note any alternative usernames, handles, or profile image names used

The alternative username or image identifier associated with 42 Tirana's IT Administrator.

**Question:** Find how many projects AADF has completed.

AADF (Albanian American Development Foundation) is a non-profit organization operating in Albania. Their official website or annual reports contain statistics on completed projects.

#### Solving Approach

1. Navigate to [https://aadf.org](https://aadf.org) or search "AADF Albania completed projects"
