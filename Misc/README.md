# Misc

## Overview

This category contains three miscellaneous challenges of escalating difficulty. Each challenge provides a ZIP archive that must be analyzed, extracted, or decrypted to obtain the flag. The topics span configuration analysis, feature flag manipulation, and key-file cryptography.

## Challenges

| Challenge | Points | File | Objective |
|---|---|---|---|
| [Config Path](./Config%20Path/) | 10 | `misc_easy.zip` | Identify and correct a wrong default configuration path |
| [Feature Toggle](./Feature%20Toggle/) | 50 | `misc_medium.zip` | Determine what disables the feature |
| [Wrong Key](./Wrong%20key/) | 100 | `misc_hard_keyfile.zip` | Bypass a decoy and extract the flag with the correct key |
