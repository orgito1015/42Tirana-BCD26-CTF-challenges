# Forensics

## Overview

This category contains five independent digital forensics challenges, each centered on a specific artifact recovered from a simulated incident. Challenges cover disk forensics, memory analysis, network packet inspection, payload recovery, and server log analysis.

## Challenges

| Challenge | Points | Artifact | Objective |
|---|---|---|---|
| [Deleted Evidence](./Deleted%20Evidence/) | 60 | `forensic_disk.img` | Recover deleted files from a disk image |
| [Initial Access](./Initial%20Access/) | 20 | `network_capture.pcap` | Identify the first malicious HTTP request |
| [Memory Secrets](./Memory%20Secrets/) | 50 | `memory_dump.raw` | Find the data exfiltration command in memory |
| [The Hidden Gem](./The%20Hidden%20Gem/) | 35 | `payload_download.pcap` | Recover a downloaded payload from the capture |
| [The Mastermind](./The%20Mastermind/) | 120 | `server_logs.tar` | Extract a flag concealed within server logs |
